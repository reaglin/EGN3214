EGN3214 — Engineering Computation

8Numerical Methods with SciPy and Curve Fitting

This module introduces learners to practical numerical computing in Python using the SciPy ecosystem. Topics span root finding, numerical integration, interpolation, linear and nonlinear curve fitting, and residuals analysis — contextualized through realistic engineering data workflows and hands-on examples.

This reading explains the concept of root finding and demonstrates how to use scipy.optimize tools (such as brentq and fsolve) to locate function zeros, covering method selection, bracketing, convergence, and practical engineering use cases where analytical solutions are unavailable.

Numerical Integration with scipy.integrate

Linear Curve Fitting with numpy.polyfit

This reading teaches polynomial and linear curve fitting using numpy.polyfit, explaining how to model noisy experimental data with best-fit lines or polynomials, interpret coefficients, assess fit quality with R², and apply the workflow to engineering datasets.

This reading covers nonlinear curve fitting using scipy.optimize.curve_fit, explaining how to define custom model functions (e.g., exponential, power-law), estimate parameters, interpret uncertainty via covariance, and apply the technique to engineering relationships that linear methods cannot capture.
