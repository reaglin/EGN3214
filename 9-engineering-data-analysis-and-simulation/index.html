EGN3214 — Module 9

Engineering Data Analysis and Simulation

This module guides you through the complete engineering data workflow—from loading and cleaning raw datasets to exploratory visualization—and introduces numerical simulation techniques for mechanical, electrical, and thermal systems using Python-based ODE solvers. Use the artifacts below to progress through readings, interactive content, and structured topic outlines.
