Module 1 — Python Environment Setup and Language Fundamentals

This module introduces the foundational tools and concepts needed to begin programming in Python. Learners will explore development environments, interpreters, package management, and core language features such as variables and assignment — building the practical foundation required for all subsequent programming work.
