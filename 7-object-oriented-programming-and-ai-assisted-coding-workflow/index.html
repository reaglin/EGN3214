EGN3214 — Module 7

Object-Oriented Programming and AI-Assisted Coding Workflow

This module introduces the principles of object-oriented programming (OOP) in the context of engineering and scientific computing, and explores how modern AI-assisted coding tools can be integrated responsibly into a professional development workflow. Work through the readings and resources below to build a strong foundation in class design, inheritance, and AI-augmented coding practices.
