Module 7 — Object-Oriented Programming and AI-Assisted Coding Workflow

This module introduces the principles of object-oriented programming (OOP) in Python and demonstrates how to model real-world engineering objects using classes, attributes, and methods. It also covers modern AI-assisted coding tools—how they work, how to craft effective prompts, and how to critically verify AI-generated code before applying it in engineering contexts.
