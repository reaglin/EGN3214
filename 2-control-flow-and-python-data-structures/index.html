Module 2 — Control Flow and Python Data Structures

This module introduces Python's core decision-making and repetition constructs— conditional statements, loops, and loop-control keywords—alongside fundamental data structures such as lists. Work through the readings, interactive activities, and quiz below to build a solid foundation in writing structured, data-driven Python programs.
