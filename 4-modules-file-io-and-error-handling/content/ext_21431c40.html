Module 4 Basic Computer Skills: Understanding File Types in Windows

Author / Site: Course Instructor (Internal Course Material)

Summary: This resource explains how file types work in Windows, covering the relationship between file formats and extensions, how to show and change extensions, and common file types encountered in programming contexts. It also provides practical guidance on creating CSV files using Notepad and working with Google Drive and Google Colab for file access.

Relevance to Module: Understanding file types and extensions is foundational for Module 4 topics such as reading and writing text and CSV files, as students must be able to locate, create, and manage files correctly before working with Python's file I/O and the with-statement context manager. The section on Google Drive and Colab directly supports workflows where students may load or save files in cloud environments while practicing file handling and data processing scripts.
