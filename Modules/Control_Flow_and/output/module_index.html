Module 2 – Control Flow and Python Data Structures

This module builds your foundation in writing programs that make decisions and repeat tasks. You will explore conditional statements (if, elif, else), for and while loops, loop control with break and continue, and core Python data structures including lists. Work through the readings, interactive activities, and quiz below to reinforce each concept.
