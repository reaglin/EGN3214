Assignment 5 – Find Depth of Flow in an Open Channel

Author / Site: Course Instructor

Summary: This assignment tasks students with using iterative numerical techniques to solve Manning's Equation for the depth of flow in a trapezoidal open channel given a target flowrate of 50 cfs. Students implement a Python function using NumPy/math libraries to compute flow area, wetted perimeter, and hydraulic radius, then iteratively converge on a depth solution with 0.01-foot accuracy. The deliverable is a Jupyter/Colab engineering report including code, a Q vs. depth plot, and documented methodology.

Relevance to Module: This assignment directly applies NumPy mathematical functions (e.g., sqrt, pow), iterative numerical computation, and function-based engineering problem solving — all core skills introduced in Module 5. It reinforces the use of NumPy as a foundation for engineering calculations by embedding these operations within a real hydraulic engineering context requiring convergence logic and result visualization.

Instructor Note

This assignment allows the student to learn how to solve problems in the format y=f(x) for x, when x cannot be explicitly solved in algebraic terms.
