  • https://docs.python.org/3/tutorial/inputoutput.html#reading-and-writing-files
  • The official Python tutorial section covering file I/O, including how to open files, use file modes, and apply the `with` statement as a context manager.
  • Explains reading and writing text files with clear examples straight from the language maintainers.

Real Python: Working With Files in Python

  • https://realpython.com/working-with-files-in-python/
  • A comprehensive tutorial covering file opening modes, reading and writing text files, and managing file paths.

Python Docs: pathlib — Object-Oriented Filesystem Paths

  • https://docs.python.org/3/library/pathlib.html
  • The official reference documentation for the `pathlib` module, detailing how to construct and manipulate file paths in an object-oriented way.
  • Covers path creation, navigation, file existence checks, and reading/writing through Path objects.

Real Python: Python's pathlib Module

  • https://realpython.com/python-pathlib/
  • A beginner-friendly deep dive into the `pathlib` module, explaining how to replace older `os.path` techniques with cleaner, more readable path operations.
  • Demonstrates common tasks like building paths, listing directories, and opening files using Path objects.

Python Docs: csv — CSV File Reading and Writing

  • https://docs.python.org/3/library/csv.html
  • The official documentation for Python's built-in `csv` module, covering `csv.reader`, `csv.writer`, `csv.DictReader`, and `csv.DictWriter` with full API details.
  • Includes explanations of dialect options, quoting behavior, and formatting parameters for fine-grained control over CSV parsing.

Real Python: Reading and Writing CSV Files in Python

  • https://realpython.com/python-csv/
  • A thorough tutorial explaining what CSV files are and how to read and write them using both `csv.reader`/`csv.writer` and the dictionary-based `DictReader`/`DictWriter` classes.
  • Provides side-by-side comparisons of the reader and writer approaches to help learners choose the right tool for their use case.

Python Docs: json — JSON Encoder and Decoder

  • https://docs.python.org/3/library/json.html
  • The official reference for Python's `json` module, documenting `json.loads`, `json.load`, `json.dumps`, and `json.dump` with full parameter descriptions.
  • Also notes the limitations of the module, including which Python types are and are not serializable by default.

Real Python: Working With JSON Data in Python

  • https://realpython.com/python-json/
  • A detailed tutorial covering JSON fundamentals, parsing JSON strings and files with `json.loads` and `json.load`, and serializing Python objects with `json.dumps` and `json.dump`.
  • Discusses practical use cases such as using JSON for configuration files and highlights common pitfalls and limitations of the standard `json` module.
