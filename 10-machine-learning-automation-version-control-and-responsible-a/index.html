Module 10 — Machine Learning, Automation, Version Control, and Responsible AI

This module introduces foundational and practical skills at the intersection of data-driven engineering and professional practice. Learners will explore supervised machine learning with scikit-learn, build Python-based automation and parameter sweep pipelines, adopt Git and GitHub for version control, and critically examine the ethical responsibilities that come with integrating AI tools into engineering workflows.
