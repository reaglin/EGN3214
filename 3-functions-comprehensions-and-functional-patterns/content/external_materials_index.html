Future Value: While Loop Input and Present Value Calculation Assignment

Author / Site: Course Instructor (Engineering Economics Module)

Summary: This assignment challenges students to write a Python program using a while loop to calculate the Present Value (PV) of equipment resale amounts and annual revenue streams across multiple cost scenarios, given a specified interest rate. Students evaluate five equipment purchase scenarios over a 10-year lifespan at a 5% interest rate, outputting a formatted table of results. The assignment also requires an engineering report analyzing investment risk under varying interest rates.

Relevance: This assignment provides a practical application context for writing functions with parameters such as interest rate, resale value, and revenue, which directly reinforces function definition and default parameter values covered in the module. The need to process multiple cost scenarios programmatically encourages use of list comprehensions or map/filter patterns to compute and collect PV results efficiently. Iterating over scenario data and generating tabular output also naturally motivates the use of loops, comprehensions, and potentially lambda functions for financial calculations.
