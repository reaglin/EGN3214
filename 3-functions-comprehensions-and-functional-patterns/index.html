Module 3 — Functions, Comprehensions, and Functional Patterns

This module explores the foundations of reusable, expressive Python code. Topics span function definitions, parameter types, *args / **kwargs, Python's LEGB scope rules, list and dictionary comprehensions, recursion, and broader functional programming patterns. Use the artifacts below to read, practice, and assess your understanding.
