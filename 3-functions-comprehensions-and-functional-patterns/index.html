Module 3: Functions, Comprehensions, and Functional Patterns

This module explores Python's function system in depth — from defining and documenting functions with def and docstrings, through variable-length arguments, scope rules (LEGB), comprehensions, and functional programming patterns such as decorators. Use the resources below to read, practise, and assess your understanding.
