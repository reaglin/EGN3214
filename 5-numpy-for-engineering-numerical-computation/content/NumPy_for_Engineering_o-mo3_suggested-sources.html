Publisher: NumPy Developers / NumPy Project

Link: https://numpy.org/doc/stable/user/basics.indexing.html

The official NumPy documentation provides authoritative, comprehensive coverage of all indexing strategies — including basic integer indexing, slicing, Boolean masking, and fancy (integer array) indexing — directly aligned with MO3. It also includes detailed reference pages for reshaping, transposing, stacking, and splitting, making it the primary go-to reference for every topic in this module.

Author: Wes McKinney

Citation: McKinney, W. (2022). Python for Data Analysis (3rd ed.). O'Reilly Media. ISBN: 978-1-098-10403-0

Author: Nicolas P. Rougier
