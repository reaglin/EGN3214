EGN3214

Module 5 — NumPy for Engineering Numerical Computation

This module introduces NumPy as the foundational library for numerical computing in engineering. Learners will explore ndarray creation and manipulation, vectorized and element-wise operations, universal functions, broadcasting, linear algebra routines, and iterative numerical methods — all applied to practical engineering problems such as open-channel flow analysis.
