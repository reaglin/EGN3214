5 – NumPy for Engineering Numerical Computation

This module introduces NumPy as a foundational tool for engineering numerical computation. Learners explore ndarray creation and manipulation, vectorized and element-wise operations, broadcasting, universal functions, linear algebra, and practical engineering applications — including iterative numerical methods such as Newton-Raphson for solving real-world problems.
